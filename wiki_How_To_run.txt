How To
Before running the application, please follow below :
1.	Make sure you put the config.properties file your Home directory (C:\Users\{USER} for Windows OS) 
2.	Open the config.properties and change the file.path values put where you have your reports.csv, reports.json, reports.xml files. 
3.	Make sur that all library are in the lib directory where the jar exe file is.
4.	2 library are principaly used : 
-	commons-csv-1.7.jar for easly csv reader, parser and writer using org.apache.commons.csv ;
-	json-simple-1.1.jar for JSON reader, parsing and writing.
-	For XML reader, parsing and writing DOM lib which is the most used.
